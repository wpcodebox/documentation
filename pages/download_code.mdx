# How to download a code snippet from the Code Snippet Repository

To download a code snippet from the Code Snippet Repository, you just need to open the snippet repository using the button above the snippet manager. You can search for the code snippet you need using either the search box or by selecting the appropriate category on the left.

Once you found the snippet you want to download, just click "Download Snippet" and the snippet will be downloaded locally. Once the snippet is downloaded, it will be disabled by default. You just need to enable the snippet to make it execute.
