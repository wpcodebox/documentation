### CDNJS Integration

Including thousands of libraries in your WordPress site is as easy as 1-2-3! Simply search for the library you need, and done - it's included.

Here's how to use the cdnjs integration:

- For Snippet Type, choose "JavaScript (External File)"
- Under "External JS File URL", choose "Load from &lt;cdn&gt;"
- You can now search the cdnjs library and load the desired library easily
