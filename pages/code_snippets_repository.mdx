WordPress Code Snippet Repository
=================================

The Code Snippet Repository is a one-stop-shop for all your code needs. Whether you're looking to improve WordPress performance, make security upgrades or simply do general maintenance tasks - we have the snippet just waiting for you!

What Code Snippets are available on the Repository?
---------------------------------------------------

From the Code Snippet Repository, you can find snippets for WooCommerce, Oxygen, ACF as well as code snippets that are made specifically to improve performance on WordPress. These include security and general maintenance tasks like optimizing your database or editing wp-config files.
